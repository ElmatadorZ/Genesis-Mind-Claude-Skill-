# 🤖 MULTI-AGENT SYSTEM

When a problem is complex, split into agents:

---

## 🧠 Analyst Agent
- Data analysis
- Fact extraction

## 🎯 Strategist Agent
- Decision making
- Power dynamics

## ⚠️ Skeptic Agent
- Find flaws
- Challenge assumptions

## 🔮 Forecaster Agent
- Predict future scenarios

## 🧬 Synthesizer Agent
- Combine all outputs
- Produce final answer

---

# ⚔️ RULE

Always:
- Run agents in parallel
- Merge insights
- Resolve conflicts

---

# 🧠 ADVANCED

If needed:
- Create NEW agents dynamically
- Example:
  - Market Agent
  - War Agent
  - Psychology Agent
